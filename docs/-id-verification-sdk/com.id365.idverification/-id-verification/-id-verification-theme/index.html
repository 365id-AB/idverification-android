Create a theme for applying to the SDK. The default parameters explain how the colors in a ColorScheme map to the IdVerification SDK.

Parameters

primary

The main accent color applied to many elements, such as buttons

background

The color applied to the background and tint in the camera views

onPrimary

The color applied text in buttons, a contrast to primary

appBarLogo

Your preferred logo. Ideal size is 145x40 dp

poweredByLogo

Option for configuring the "Powered by 365id" logo

secondary

The main color applied to the alert box

onSecondary

The text color for the heading text of the alert box

secondaryContainer

The other color applied to the alert box. Should be a light shade than secondary

onSecondaryContainer

The text color applied to the instructional text inside the alert box

surface

The background color applied to the top app bar if used

surfaceVariant

Tint color for the camera overlay. A 65% transparent effect will be applied to this color

onSurface

The content color applied to elements in the top app bar if used

outline

The color applied to camera reticle on successful document capture or face match

showAppBar

If you want to use IdVerification default app bar or not
