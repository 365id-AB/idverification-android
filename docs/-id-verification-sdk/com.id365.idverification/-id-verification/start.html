fun start(ctx: Context, request: IdVerificationRequest, skipModules: List<IdVerification.SkipModule> = emptyList(), documentSizeType: IdVerification.DocumentSizeType = DocumentSizeType.DOCUMENT, eventHandler: IdVerificationEventHandler): Boolean

This is the main entry point for the 365ID Scanner SDK for Android. This should be called before embedding or navigating to the IdVerificationView.

Return

true if the SDK was initialized successfully, otherwise false.

Parameters

ctx

: An ApplicationContext. Should not be an ActivityContext.

request

: An IdVerificationRequest. Should contain a valid token and a location (Name or ID).

skipModules

: Modules that can be skipped during the identification process.

documentSizeType

: The type of document that is supposed to be scanned (ID1, ID3 or DOCUMENT). Only impacts the viewfinder size.

eventHandler

: Implementation of IdVerificationEventHandler that reports to implementor when certain events happen
